**Kiet Magazine** was developed to solve the problem of scattered digital content. In a world where students and professionals have to visit multiple websites for news, research, and lifestyle reading, this project offers a **single, centralized hub**.

I built this project to demonstrate how frontend technologies (HTML, CSS, JS) can create a seamless, app-like experience in a web browser. It features a secure login gate, a responsive grid system that handles images perfectly, and a backend-free feedback loop using API integration.
